GitHub Star 造假？

撕逼 Star 发图？

互怼对方人品？

荒谬。

丢人丢到国外去。

虽然不知道真实发生了什么；

但我觉得，把 GitHub 变成一个风口浪尖的舆论场地，不合适。

也没必要追究事情到底是怎样；要吵，去新闻媒体，去找吃瓜群众，别来污染友好学习的社区。

乌烟瘴气。

希望自己能够保持远离舆论，坚持技术纯粹、技术无罪的态度吧。

做好自己，维护好自己的博客、项目；

不断学习，继续加油。
